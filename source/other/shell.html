SHELL¶

Shellの種類¶

  • shBourne shell

    ベル研究所のStephen R. Bourneが開発したシェル。B shellとも。

  • cshC shell

    カリフォルニア大学バークレイ校のBill Joyが開発したシェル。

  • kshkorn shell

    ベル研究所のDavid Kornが開発したシェル。Bourne shellとC shellの長所を程よく取り入れていた。

  • bashBourne Again SHell

    Free Software Foundationが開発したシェル。

  • tcsh :

    C shellを改良したシェル。

  • zsh :

    Paul Falstadがプリンストン大学の学生であった時に開発したシェル。

Z shell¶

Install¶

yum install -y zsh

cat /etc/shells # Show Current Shell

chsh -s /bin/zsh # Change Current Shell

vi .zshrc

# .zshrc ---------------------------------

autoload -U compinit
compinit

# .zshrc ---------------------------------
