用語¶

仮想化において頻出の用語を下記に記載する。

  • P2V

    Phisical to Virtual

  • V2V

    Virtual to Virtual

  • Live migration

  • Para-Virtualization

  • Full Virtualization

  • バイナリトランスレーション

  • CPU仮想化支援機能 CPUエミューレーション CPU仮想化支援機能の実装として代表的なCPUベンダーは以下のテクノロジーを提供している。 Intel VT (Intel Virtualization Technology) AMD-V(AMD Virtualization)

  • CPU Flags(フラグ) CPUがサポートしている機能を示すフラグ（目印）のことであり、Linuxシステムでは/proc/cpuinfoファイルに情報が格納されている。 CPUの仮想化支援機能が利用できる場合は/proc/cpuinfoファイルのflags行に「vmx」(Intelの場合)もしくは「svm」(AMDの場合)という文字列が記述されます。

Warning

CPUフラグはあくまでもCPUがその機能をサポートしているかどうかを表すだけのものであり、BIOSレベルでその機能が有効化されていることを保証するものではありません。
